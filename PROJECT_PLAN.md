# Space Rocket Game 기획서

## 1. 대회 적합성

`Space Rocket Game`은 월간 해커톤 “10분 안에 중독시켜라 - 웹 미니게임 챌린지”의 성격에 맞춰 설계한 짧은 세션형 브라우저 게임이다. 별을 먹어 점수를 올리고, 점수가 오를수록 로켓 속도와 장애물 압박이 커지는 구조로 즉시 이해 가능한 규칙과 반복 도전 동기를 제공한다.

## 2. 게임 콘셉트

- 장르: 흑백 레트로 아케이드 생존 게임
- 핵심 목표: 정사각형 우주 보드 안에서 별을 먹으며 가능한 오래 생존
- 조작 방식: 방향키 또는 WASD
- 플레이 감각: Snake의 제한된 보드 긴장감과 Dodge 게임의 회피 재미 결합

## 3. 핵심 재미 설계

### 빠른 이해

플레이어는 로켓, 별, 장애물만 보면 바로 규칙을 이해할 수 있다. 별은 점수, 우주 쓰레기와 소행성은 위험, 반짝이는 별똥별은 회복이라는 의미를 흑백 도형만으로 구분한다.

### 상승 곡선

별을 먹을수록 점수가 올라가고 로켓 속도가 빨라진다. 우주 쓰레기는 점수 구간에 따라 등장 빈도와 동시 등장 수가 증가하며, 100점 이후에는 더 큰 소행성이 추가되어 회피 판단을 어렵게 만든다.

### 리스크와 보상

일반 별똥별은 보드 모서리에서 반대 모서리까지 대각선으로 지나가며 맞으면 목숨이 0.5 감소한다. 반짝이는 별똥별은 더 드물게 등장하고 먹으면 목숨 1개를 회복한다. 같은 대각선 패턴을 공유하므로 플레이어는 위험과 보상을 순간적으로 판별해야 한다.

## 4. 주요 규칙

- 별을 먹으면 점수 +1
- 별은 항상 1개만 존재
- 벽, 우주 쓰레기, 소행성에 닿으면 목숨 -1
- 일반 별똥별에 닿으면 목숨 -0.5
- 반짝이는 별똥별에 닿으면 목숨 +1, 최대 3
- 목숨이 0이 되면 Game Over
- 점수가 높아질수록 로켓 속도, 장애물 속도, 장애물 등장 빈도가 증가

## 5. 화면 구성

- Start: 게임명, 조작법, 시작 안내
- Play: 정사각형 보드, SCORE, LIFE
- Pause: 일시정지 안내
- Game Over: 최종 점수와 재시작 안내
- End: 장기 목표 달성 화면

## 6. 기술 설계

- Vite
- TypeScript
- HTML Canvas
- 외부 이미지 없이 Canvas 도형만 사용
- 흑백 계열만 사용하는 미니멀 그래픽
- 모듈 구조:
  - `Game.ts`: 게임 루프, 상태, 점수, 생명, 난이도
  - `Player.ts`: 로켓 이동과 렌더링
  - `Star.ts`: 별 생성과 획득
  - `Obstacle.ts`, `Asteroid.ts`: 장애물
  - `Background.ts`: 배경 별, 일반/회복 별똥별
  - `HUD.ts`, `screens.ts`: UI

## 7. 차별점

- Snake 스타일의 보드 제한을 로켓 생존 게임으로 재해석
- 점수 상승이 보상인 동시에 속도 상승 리스크로 작동
- 별똥별을 단순 배경 효과가 아니라 반 칸 피해와 회복 아이템으로 확장
- 흑백만 사용해 해커톤 제출물의 컨셉 일관성과 완성도 강화

## 8. 향후 개선

- 최고 점수 저장
- 모바일 터치 조작
- 효과음과 음소거 옵션
- 난이도 프리셋
- 점수별 배경 밀도 변화
