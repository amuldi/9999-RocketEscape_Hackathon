Rocket Escape

검은 우주 보드 안에서 빨라지는 로켓으로 별을 먹고, 우주 쓰레기와 소행성, 별똥별을 피하는 흑백 고속 생존 미니게임.

Deploy: https://space-rocket-game-smoky.vercel.app

1. 프로젝트 개요

9999: Rocket Escape는 Snake 게임의 제한된 공간 긴장감을 로켓 회피 액션으로 재해석한 브라우저 미니게임이다. 플레이어는 정사각형 우주 보드 안에서 로켓을 조종해 별을 먹고, 점점 빠르게 날아오는 장애물을 피한다.

점수 획득은 단순한 보상이 아니라 난이도 상승의 시작점이다. 별을 먹을수록 로켓이 빨라지고, 동시에 장애물 속도와 등장 빈도가 증가한다. 그래서 짧은 플레이 안에서도 속도감, 긴장감, 재도전 동기가 빠르게 형성된다.

한 줄 콘셉트
검은 우주 보드 안에서 빨라지는 로켓으로 별을 먹고, 장애물을 피해 최고 점수에 도전하는 흑백 고속 생존 게임.

2. 대회 적합성

웹 미니게임 챌린지는 사용자가 짧은 시간 안에 규칙을 이해하고 바로 반복 플레이할 수 있는 구조가 중요하다. 이 게임은 별, 로켓, 장애물이라는 직관적인 요소만으로 목표와 위험을 즉시 전달한다.

  • 첫 화면에서 조작법을 확인하고 바로 시작할 수 있다.
  • 별을 먹으면 점수가 오르는 규칙이 단순하다.
  • 점수가 오를수록 속도가 올라 체감 난이도가 즉시 변한다.
  • 목숨 3개와 짧은 무적 시간으로 실패 후 재도전 흐름이 빠르다.

3. 핵심 재미 설계

빠른 이해

플레이어는 로켓, 별, 장애물만 보고도 규칙을 바로 이해할 수 있다. 별은 점수, 우주 쓰레기와 소행성은 위험, 반짝이는 별똥별은 회복이라는 의미를 도형과 움직임으로 구분한다.

상승 곡선

점수가 오르면 로켓 속도와 장애물 속도가 함께 올라간다. 플레이어가 성장하는 동시에 위험도 커져 계속 집중하게 만든다.

리스크와 보상

일반 별똥별은 피해를 주지만, 반짝이는 별똥별은 목숨을 회복한다. 같은 대각선 패턴 안에서 위험과 보상을 순간적으로 판단해야 한다.

속도 체감

빠른 장애물, 별똥별, 로켓 불꽃, 보드 내부 스피드 라인이 함께 움직여 단순 수치 이상의 고속 플레이 감각을 만든다.

4. 주요 규칙

요소 규칙
별 항상 1개만 존재하며, 먹으면 점수 +1 후 새 위치에 생성된다.
로켓 정사각형 보드 밖으로 나갈 수 없고, 벽 충돌 시 목숨이 감소한다.
목숨 최대 3개. 일반 충돌은 -1, 별똥별 충돌은 -0.5다.
회복 반짝이는 별똥별을 먹으면 목숨이 1 회복된다.
클리어 9999점에 도달하면 END 화면을 표시한다.

5. 장애물과 난이도

우주 쓰레기

  • 게임 초반부터 등장하는 기본 장애물이다.
  • 화면 바깥에서 생성되어 보드 안을 직선으로 가로지른다.
  • 점수 70점 이후부터 한 번에 최대 2개까지 등장한다.
  • 점수 150점 이후부터 한 번에 최대 3개까지 등장한다.
  • 점수가 높아질수록 이동 속도와 등장 빈도가 함께 증가한다.

소행성

  • 100점 이상부터 등장한다.
  • 우주 쓰레기보다 크고 더 빠르다.
  • 고득점 구간에서 플레이어의 이동 경로를 크게 압박한다.
  • 불합리한 즉사 패턴을 줄이기 위해 플레이어 근처를 직접 관통하는 생성은 피한다.

별똥별

  • 보드 모서리에서 반대쪽 모서리까지 대각선으로 이동한다.
  • 일반 별똥별은 가장 빠른 장애물보다 더 빠르게 이동한다.
  • 위험 별똥별은 매우 가끔 등장하고, 맞으면 목숨이 0.5 감소한다.
  • 회복 별똥별은 더 드물게 등장하고, 먹으면 목숨이 1 회복된다.

6. 시각 디자인

전체 디자인은 흑백만 사용한다. 검은색 보드는 우주 공간을 의미하고, 흰색 계열 도형은 로켓, 별, 장애물, UI 정보를 표현한다. 외부 이미지 에셋 없이 Canvas 도형만으로 제작해 레트로하고 일관된 인상을 만든다.

  • 검은색 정사각형 보드와 흰색 테두리
  • 흰색 로켓 실루엣과 불꽃 파티클
  • 작은 배경 별, 속도선, 대각선 별똥별
  • SCORE와 LIFE 중심의 간결한 HUD

7. 화면 구성

화면 역할
Start 게임명, 조작법, 시작 안내를 보여준다.
Play 정사각형 보드, 로켓, 별, 장애물, SCORE, LIFE를 표시한다.
Pause 일시정지 상태와 재개 안내를 표시한다.
Game Over 최종 점수와 재시작 안내를 보여준다.
END 9999점 도달 시 클리어 메시지를 표시한다.

8. 기술 구조

프로젝트는 Vite와 TypeScript를 기반으로 하며, HTML Canvas에서 게임 루프와 렌더링을 직접 제어한다. 기능별 폴더를 분리해 게임 흐름, 오브젝트, 이펙트, UI를 쉽게 파악할 수 있도록 구성했다. 

src/
  main.ts
  game/
    core/       Game loop, input, collision, difficulty, shared types
    objects/    Player, star, debris, asteroid
    effects/    Background, meteors, flame particles
  ui/           HUD and state screens
  styles/       Global CSS

9. 차별점과 확장 방향

  • Snake의 제한 공간 규칙을 로켓 생존 액션으로 변형했다.
  • 점수 획득이 속도 상승으로 이어져 보상과 위험이 동시에 커진다.
  • 별똥별을 단순 배경 효과가 아닌 피해와 회복 시스템으로 확장했다.
  • 향후 최고 점수 저장, 모바일 터치 조작, 효과음, 랭킹 보드로 확장할 수 있다.
